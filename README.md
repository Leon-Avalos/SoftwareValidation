# SoftwareValidation - CRUD Estudiantes

### Como ejecutar el proyecto?

1. Instalar paquetes 
```bash
npm install
```
2. Ejecutar proyecto en entorno local
```bash
npm run serve
``` 

### Como ejecutar los tests

Se ejecutaran los tests declarados en tests/server.spec
```bash
npm run test
```
